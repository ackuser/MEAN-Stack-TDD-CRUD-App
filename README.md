# MEAN-Stack-TDD-CRUD-App
Mongo Express Angular Node - TDD - with Karma, Protractor, & Jasmine

###Install:
  `npm install`

###Start server: 
  `node server.js`

####Testing
#####Karma:
  `npm test`
  
#####Protractor:
  `npm run protractor`
  or
  `protractor test/protractor-conf.js`

######Note: 
  To not error with Protractor add one widget prior to running.
